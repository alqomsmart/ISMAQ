Panduan Pintar SIPONTREN

Selamat datang di Sistem Informasi Pondok Pesantren (SIPONTREN). Aplikasi ini berjalan sepenuhnya di browser Anda. Berikut panduan singkat:

  • Data Santri: Tambahkan data santri baru dan cetak Kartu QR Code mereka di menu ini.
  • Jadwal Kegiatan: Atur sesi kegiatan (misal: Subuh, Ashar) sebelum menggunakan fitur absensi.
  • Scan Absensi: Gunakan kamera laptop/HP atau input manual untuk absensi. Notifikasi WA otomatis ke orang tua jika token Fonnte diisi.
  • Setor Hafalan: Catat hafalan Qur'an atau kitab santri dan cetak sertifikat (Syahadah).
  • Pengaturan: Jangan lupa melakukan Backup Data secara berkala (Cloud atau Download Lokal).
